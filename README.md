#### 프로젝트 소개 

> 최근 환경 보호에 대한 관심이 증가함에 따라 환경보호에 대한 정보를 자유롭게 나눌 수 있는 **제로웨이스트 실천, 쇼핑, 환경 교육을 위한 통합플랫폼**을 기획, 제작하였습니다.

> 개발기간 : 2023/07/24 - 2023/08/18

> 구현 기능 <br>
상품리뷰, 문의글과 관련된 CRUD기능과 상품조회, 정렬, 좋아요, 페이징기능 및 장바구니 추가, 선택 및 품절상품 삭제 기능
